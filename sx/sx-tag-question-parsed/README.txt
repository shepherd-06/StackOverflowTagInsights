Folder `parsed` is a collection of 168 Stack Exchange tag-question bipartite networks. The
files look as follows.

$ head -5 emacs.txt
-------
14 1 1 2014-09-23T18:18:10.547
24 2 2 2014-09-23T18:38:48.913
39 3 7 2014-09-23T19:37:55.753
39 4 7 2014-09-23T19:37:55.753
39 5 7 2014-09-23T19:37:55.753
-------

Column information:  
1. ID of user asking question.
2. ID of tag.
3. ID of question (post).
4. Timestamp of question.
